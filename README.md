# OrdinancaPro - Sistemi i Menaxhimit te Ordinances

Sistem i plote per menaxhimin e pacienteve, terapive dhe raporteve financiare.

## Instalimi dhe Nisja
1. Shkarkoni kodin.
2. Run `npm install` per te instaluar dependencat.
3. Run `node server.js` per te nisur serverin.

## Teknologjite
- Frontend: React.js
- Backend: Node.js (Express)
- Database: JSON (Local File)
